Program description

Movies Interface is a tiny web software which allows you to stream videos stored in your computer on the local network.

Movies Interface was created on PHP. Therefore PHP core software must be installed.

I recommend installing XAMPP. A Wizard user interface that installs and manipulates Apache2 service (PHP core).

Note
Movies Interface uses HTML5 web player which means only the following formats are supported:

  • mp4
  • WebM
  • ogg
  • mov
  • mkv (no sound)
If a desired format not listed is wished to be used, you can also use Movies Interface to download mentioned videos on the device that enters Movies Interface.

Please, go to Movies Interface's Github repository to learn how to setup Movies Interface.

Program features
  • Feather size.
  • Relatively easy to configure.
  • Light local network bandwidth consumption.
  • Compatible with most browsers.
  • Responsive to screen size (can be used on mobile devices).
  • Direct links to download movies you add to the software.
