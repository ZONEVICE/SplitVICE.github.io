Program description

This program is able to emulate keyboard keystrokes making it compatible on ANY software. Specifically, the ones commonly used to save your progress/work on games, office documents, media or digital software, etc. such as CTRL + S, CTRL + G or F2 (some game emulators).
You are able to set a timer in-between every save action.

Current saving shortcuts:
Version 1.3 and 1.3.1: CTRL + G. CTRL + S. CTRL + A. CTRL + Shift + S. F1. F2. F3. F4. F5. F6. F7. F8. F9. F10. F11. F12.
Version 1.2: CTRL + G. CTRL + S. CTRL + A. F1. F2. F3. F4. F5. F6. F7. F8. F9. F10. F11. F12.
Version 1.0 / 1.1: CTRL + G. CTRL + S.

Program features
  • Alert message when the program has done a save action.
  • Choose between 5 different positions to display the alert save message.
  • Works on any software, game, office document.
  • A timer to configure saving actions.
  • Remembers your configuration.
  • Extremely light.
  • Easy to use.
